# Portfolio — Ayman El Karroussi

> Full Stack Software Engineer | Java · Spring Boot · Cloud

**Live:** [ayman-portfolio.vercel.app](https://ayman-portfolio.vercel.app)

---

## Sections

| Section | Description |
|---|---|
| Hero | Positioning, CTA buttons, CV download |
| À propos | Profile summary, quick facts |
| Compétences | Categorized skill grid with level indicators |
| Projets | Project cards — MyCampo, PhableAI, SUPFile, Portfolio |
| Parcours | Vertical timeline — education, experience, certifications |
| Contact | Direct links — email, LinkedIn, GitHub, CV |

---

## Stack

| Layer | Technology |
|---|---|
| Frontend | React 18, Vite 6 |
| Styling | CSS Modules, CSS custom properties |
| Fonts | IBM Plex Mono, DM Sans (Google Fonts) |
| Deployment | Vercel |
| Version control | Git / GitHub |

---

## Run locally

```bash
npm install
npm run dev
```

App available at `http://localhost:5173`

## Build

```bash
npm run build
```

Output in `dist/`.
